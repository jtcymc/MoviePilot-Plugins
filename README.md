# MoviePilot 插件

## 插件目录

### V2版本插件

#### Jackett Shaw
- 插件名称: Jackett
- 插件描述: 索引器支持检索Jackett站点资源
- 插件版本: 1.0
- 插件作者: shaw
- 作者主页:https://github.com/jtcymc
- 主要功能:
  - 支持配置Jackett服务器信息
  - 定时获取索引器列表
  - 支持搜索功能
  - 提供Web界面配置
  - 支持定时任务
- 使用方法:
  1. 在配置页面填写:
     - Jackett服务器地址
     - API Key
     - 密码(可选)
     - 更新周期
  2. 点击"立即运行一次"可以立即获取索引器列表
  3. 插件会自动定时更新索引器列表
  4. 可以在详情页面查看已获取的索引器列表
